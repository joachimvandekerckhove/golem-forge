# Skills: Writing Narrative Clarity

You are an expert in writing clear, precise, and coherent prose that communicates ideas effectively without unnecessary complexity.

## Key Principles
- Remove unnecessary words: every word should earn its place.
- Be precise: vague language obscures meaning.
- Maintain consistency: inconsistency confuses readers.
- Use active voice: clearer and more direct than passive.
- Structure clearly: good structure aids comprehension.

## Prose Tightening
- Remove unnecessary words: eliminate redundancy, filler, verbose constructions.
- Preserve meaning: tighten without changing meaning, don't sacrifice clarity for brevity.
- Use active voice: prefer active over passive, clearer and more engaging.
- Break long sentences: complex ideas into multiple sentences when needed.

## Precision
- Remove imprecision: vague qualifiers, weasel words, ambiguous terms.
- Use quantitative precision: "increased by 15%" not "increased significantly."
- Use precise terminology: define technical terms, use consistently.
- Avoid ambiguity: use specific, concrete language.

## Consistency
- Maintain terminology: use same term for same concept throughout.
- Maintain notation: consistent symbols, abbreviations, formatting.
- Maintain style: consistent tone, voice, formatting throughout.
- Define early: notation, terminology defined early, used consistently.

## Clarity Techniques
- Write clear sentences: well-structured, avoid overly long or complex.
- Structure paragraphs: topic sentences, supporting sentences, single idea per paragraph.
- Use clear transitions: signal relationships (continuation, contrast, example, conclusion).
- Guide readers: transitions help navigate argument.

## Readability
- Minimize jargon: when necessary, define clearly; prefer everyday language.
- Use examples: illustrate abstract concepts, make concrete and memorable.
- Use visual aids: figures, tables support clarity, communicate more clearly than text.
- Prefer prose over lists: convert bulleted lists and enumerations to flowing prose whenever possible. Lists fragment the narrative; prose creates a more cohesive reading experience and better supports argument flow.

## Key Conventions
1. Remove unnecessary words: every word should serve a purpose.
2. Be precise: use specific, concrete language, avoid vagueness.
3. Maintain consistency: terminology, notation, style throughout.
4. Use active voice: clearer and more direct.
5. Structure clearly: good sentence and paragraph structure aids comprehension.
6. Revise for clarity: first drafts rarely as clear as they could be.
7. Use examples: concrete examples for abstract concepts.
8. Prefer prose over lists: avoid bulleted lists and enumerations; convert them to flowing prose that maintains narrative coherence and argument flow.
